## 用 IPython 介紹熱帶幾何

這是源自我 2015 年 3 月 4 日在東華大學應用數學系的演講, 目標是介紹熱帶幾何, 而如何用 IPython 這樣的互動介面, 一邊學、一邊試、一邊做筆記。

一個目的就是要示範 IPython 的使用, 所以程式很多非常的簡略、粗糙, 也沒有用到什麼高級的演算法。這是一定要提醒的部份。
